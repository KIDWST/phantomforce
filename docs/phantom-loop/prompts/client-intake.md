# Client Intake Prompt

Use this prompt when Phantom needs to turn a raw client goal into structured inputs.

## Goal

Collect only the information needed to draft a safe build packet.

## Required Output

- User goal
- Business type
- Audience
- Desired outcome
- Existing assets or links
- Missing inputs
- Risk notes
- Approval needed before outside-world action

## Safety

Do not publish, send, deploy, charge, connect accounts, or access private data.
