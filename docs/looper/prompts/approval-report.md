# Approval Report Prompt

Use this prompt when a human needs to decide whether a drafted action can move forward.

## Required Output

- Action requested
- Files or records affected
- External systems touched
- Cost or credit impact
- Rollback or undo path
- Risk level
- Recommended approval decision

## Rule

If the action sends, publishes, deploys, charges, connects accounts, or writes production data, require explicit human approval.
